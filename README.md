# 🎬 Movie Review Sentiment Analysis

## 📌 Overview
This project performs **sentiment analysis on movie reviews** using deep learning models. The objective is to classify reviews as **positive** or **negative** based on textual data.

Two different models are implemented and compared:
- Vanilla LSTM
- Bidirectional LSTM (BiLSTM)

---

## 🚀 Features
- Text preprocessing using NLP techniques
- Tokenization and sequence padding
- Deep learning-based text classification
- Comparison between LSTM and BiLSTM models
- Model evaluation using accuracy and loss

---

## 🧠 Models Used

### 1. Vanilla LSTM
- Processes text in a single direction (left → right)
- Captures past context only
- Faster and simpler

### 2. Bidirectional LSTM
- Processes text in both directions (left → right and right → left)
- Captures richer context
- Generally gives better performance
